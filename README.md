# Shadow Strike

Shadow Strike is a futuristic tactical shooter game website built using HTML, CSS, and JavaScript. The project features an immersive user interface, interactive agent profiles, dynamic battle maps, and a modern gaming-inspired design.

## Features
- Interactive Home Page
- Hologram Agent Showcase
- Dynamic Maps Explorer
- Battle Statistics Dashboard
- News & Updates Section
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub
